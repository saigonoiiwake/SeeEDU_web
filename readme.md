# SeeEDU

## Goal
Platform for SeeEDU

## Framework 
Laravel 5.5.40

## Requirement
- Composer
- PHP >= 7.0.0
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension

## Run

1. Install Dependency
```shell
composer install
```

2. Do Migration
```shell
php artisan migrate
```

3. Run Server
```shell
php artisan serve
```
